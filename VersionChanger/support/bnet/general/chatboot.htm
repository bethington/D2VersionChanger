    Help, I'm getting booted from the chat rooms

    • This is often caused by a poor connection to the Internet. If this occurs randomly, reboot your computer then try connecting to Battle.net again.
    • If the problem persists, delete the file BNCACHE.DAT in the folder you installed the game to. If this file gets corrupted it can cause problems of this nature.
    • Next Delete the contents of the "TEMP" folder inside your windows directory (normally c:\windows\temp).
    • Check the Windows Update page to make sure you have the latest Windows updates.
