    Non AOL Users:

    • Please check the latency information page.

    AOL Users:

    • Download and install the latest version of AOL:
      • Connect to AOL
      • Click on the Keyword button
      • Type "UPGRADE" then click GO
      • Select the latest AOL version available.
      • Follow the Online Instructions

    • Make sure you are connecting to AOL with parental controls turned off.

    • If you have continuing problems try logging onto AOL using the Main Account.

    • You can also try connecting to AOL using a different local access number.

    • If problems continue please check the Latency information page.
