    Windows updates
      Microsoft Security Updates:
      • Check Microsoft's web site at http://www.microsoft.com/security/default.asp for any new security fixes.

      To determine the version of Windows you are using:

      • Click on "Start"
      • Click on "Settings"
      • Click on "Control Panel"
      • Double click on the "System" icon
      • The version should be listed under Microsoft Windows.

      All Windows Versions:

      • Check the Windows updates page for any updates that apply to your system.

      All Windows 95 versions:

      • Regclean is designed to clean up unnecessary registry entries. Security fixes for a couple common attacks
        • Out-of-Band attacks go to the Microsoft search page and look for "Update to Windows 95 TCP/IP to Address Out-of-Band Issue".
        • Corrupted Internet Control Message Protocol (ICMP) datagram fragment attacks goto the Microsoft search page and look for "Invalid ICMP Datagram Fragments Hang Windows NT, Windows 95".

      Windows NT:

      • Users should make sure they have the most recent Service Pack for NT installed.
      • We also recommend installing any of the Microsoft Hot Fixes that apply to your computer.
      • Make sure to install all the security updates that apply to your computer to help prevent remote users from crashing your system over an Internet connection.
