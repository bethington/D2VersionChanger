    Most video and sound problems can be solved by the following steps:
    • Contact your computer manufacturer or video and sound card manufacturer (if purchased seperately) to find out what version of Direct X their latest drivers support.
    • Download and install the version of Direct X they suggested.
    • Run the video test program and use the default setting.
    • If problems persist, run the video test program and set the video mode to direct draw.

    Steps to run the Video Test program to select the video mode:

    • click start
    • Select Programs
    • Select Diablo II
    • Click on Video Test

    Video Modes supported by Diablo II:

      Diablo II supports Glide™ and Direct 3D™ compatible video cards with at least 8MB of video RAM.
      Direct 3D™ requires 64 MB of system RAM.
      Diablo II's resolution is set at 640x480, while the Battle.net chat interface is set at 800x600.

    Skipping Sound:

    • Click start, settings, control panel, then double click on Multimedia.
    • Make sure "Preferred Device" is set to your sound card.
    • Make sure "use only preferred devices" is checked.

    Sounds do not match the character animations in the choose a class menu:

    • Click start, settings, control panel, then double click on Multimedia.
    • Make sure "use only preferred devices" is checked.
    • Adding more RAM may also fix this problem.
