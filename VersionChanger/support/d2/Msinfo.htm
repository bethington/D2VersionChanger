Microsoft System Information (msinfo32)



    Windows 95:
    • Make sure the Diablo II CD is in the CD-ROM drive
    • Click Start, Find, Files or Folders
    • Next to Named type "msinfo32.exe"
    • Next to Look in select "C drive (C:)" or "Local hard drives (x:, x:, ...)"
    • Click Find Now to search for the file, double click on the file to run it
    • If you can't find the msinfo32.exe file:
      • Open any folder, click view, options or folder options, view
      • "Hide file extensions for known file types" Should not be checked
      • "Show all Files" Should be checked
      • Click apply and Ok, then re-search for msinfo32.exe
    • Click File, Save
    • You will see a box saying where the file was saved and with what name.
    • Attach that text file to an e-mail to support@blizzard.com along with the information requested on the contact page.

    Windows 98:

    • Make sure the Diablo II CD is in the CD-ROM drive
    • Click Start, Programs, Accessories, System Tools, System Information.
    • Select file, then Export.
    • Attach that text file to an e-mail to support@blizzard.com along with the information requested on the contact page.

    Windows 2000:

    • Make sure the Diablo II CD is in the CD-ROM drive
    • Click Start, and Choose Run
    • Choose Run
    • Type: msinfo32
    • Click OK
    • Select Action
    • Save As Text File...
    • Type in the filename and choose Save
    • Attach that text file to an e-mail to support@blizzard.com along with the information requested on the contact page.
