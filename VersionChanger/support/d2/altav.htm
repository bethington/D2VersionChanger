    When the Cinematic starts I get an unhandled exception error

      Some programs constantly run in the foreground of the Windows desktop. An example of this is Alta Vista's free internet service ad banner. When Diablo II tries to run at full screen, it can conflict with these programs and cause an Unhandled Exception. Before starting Diablo II, try closing all applications running in the background.


    Steps to close programs running in the background:

      Push the <Ctrl>, <Alt>, and <Delete> keys all at the same time. This will bring up the close program window. End task on all programs except Explorer and Systray.
      Note: These programs will restart when you reboot your computer.


    Steps to modify your startup files:

    • Click start, run, then type "msconfig" and push enter.
    • Select the Startup tab
    • Uncheck all the boxes and click OK.
    • When the "System Settings Change" window appears, click ok to restart the computer.
      Note: To start these programs again all you need to do is run "msconfig", recheck the boxes, and reboot your computer.
