    What is a Hardcore game?

      A Hardcore game is the same as a normal game untill your character dies. Once a Hardcore character dies it will be dead forever, and thus can no longer create or join games. This character will be given the Grey Robed Figure in the Battle.net chat channels.


    How do I play Hardcore games?

      First you need to create a Hardcore character. In order to do this you need to complete Diablo II with one of your normal characters. After you have completed Diablo II, the next time you create a character there will be a check box labeled Hardcore. If you select this character to be Hardcore, they will only be able to join or create Hardcore games.
